Hello there! I'm Philipnah!

My Coding expertise

Breif version:
I primarily code in Python since it's the language I'm the most comfortable in. I also code in HTML and CSS, and I'm learning a bit of PHP and MySQL from my friend.

Long version:
I started coding HTML and CSS back in 2016, I had just moved back from France and I found some video tutorials on YouTube with a very good teacher.
I was very excited about all the thing I could do with this newly acquired "skill", but I didn't know that I was actually trying to find a programming language like Python.
I knew a bit about JavaScript, but I didn't really want learn it, since I didn't really see the purpose in it. (Or perhaps because I was lazy)
Then in the start of 2018, my cousin Maja, a datalogist, showed me some simple Python and I found my programming spark again! I started coding a whole lot after that and I became really good at Python (according to me).
Now here I am, with my acquired skills, ready to learn more and improve both as a programmer and as a person!

What is my real name and What do I do?
